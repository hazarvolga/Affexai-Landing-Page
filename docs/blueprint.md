# **App Name**: AffeXAI: AI Document Intelligence

## Core Features:

- Smart Document Extraction: Extracts text, tables, fields, signatures from invoices, forms, PDFs, and images using AI.
- AI-Powered Validation: Cross-checks fields, detects inconsistencies, and highlights anomalies using AI validation.
- Real-Time Classification: 99% accurate document type recognition and auto-routing into workflows powered by AI.
- Developer-Friendly API: REST API, Webhooks, SDKs, Cloud or on-premise deployment.
- Interactive Demo: Showcase a document on the left and structured JSON output on the right with green highlight animation showing recognized fields.
- Pricing Cards: Display three pricing tiers: Free, Pro, and Enterprise, highlighting the benefits of each.

## Style Guidelines:

- Primary BG Color: Deep Dark Gray (#1A1A1A) that can subtly gradient into #0F0F0F for depth, creating a premium, futuristic aesthetic.
- Primary Accent Color: Strong neon green (#39FF14) representing intelligence and digital glow. Use for key elements like primary CTA buttons and highlighted keywords.
- Secondary Accent: Neon yellow (#EFFF00) for secondary CTAs, badges, highlights, alert messages, and subtle lighting effects.
- Text Colors: White (#FFFFFF) for titles, Light Gray (#CCCCCC) for paragraph text, and #39FF14 / #EFFF00 for highlighted keywords.
- Professional but accessible, future-oriented but grounded in real utility. Emphasize automation, accuracy, and speed with short, powerful sentences.
- Use neon green glowing icons to represent key features and benefits.
- Implement a 4-12 column responsive grid with generous spacing.
- Incorporate smooth animations and microinteractions for a high-tech feel.
