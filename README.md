# AffeXAI Landing Page

A modern, responsive landing page for AffeXAI - an AI-powered document intelligence platform.

## Features

- Built with Next.js 15 and React
- TypeScript for type safety
- Tailwind CSS for styling
- Responsive design for all device sizes
- Dark mode support
- Framer Motion for animations
- Genkit AI integration
- Firebase integration

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

### Build

```bash
npm run build
```

### Deployment

This project can be deployed to Coolify, Vercel, Netlify, or any other hosting platform that supports Next.js applications.

## Project Structure

```
src/
├── app/              # Next.js app directory
├── components/       # React components
├── lib/              # Utility functions
├── hooks/            # Custom hooks
└── ai/               # AI-related functionality
```
